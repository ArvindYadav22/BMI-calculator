# BMI Calculator

A modern, responsive BMI Calculator web application.

## Features

-   **Modern UI**: Clean, glassmorphism-inspired design with smooth gradients and animations.
-   **Responsive**: Works perfectly on desktop and mobile devices.
-   **Dynamic Feedback**: Instantly calculates BMI and provides a category (Underweight, Normal, Overweight).
-   **Visual Indicators**: Color-coded results to easily interpret your BMI.

## How to Run

You can run this project in two ways:

### Option 1: Direct Open (Simplest)
1.  Navigate to the project folder on your computer.
2.  Double-click the `index.html` file.
3.  The application will open in your default web browser.

### Option 2: Local Server (Recommended for Development)
If you are using VS Code:
1.  Install the "Live Server" extension.
2.  Right-click `index.html` and select "Open with Live Server".
3.  The app will launch at `http://127.0.0.1:5500/index.html` (or similar).


## Usage

1.  Enter your height in centimeters (cm).
2.  Enter your weight in kilograms (kg).
3.  Click "Calculate BMI".
4.  View your BMI score and category below the button.
